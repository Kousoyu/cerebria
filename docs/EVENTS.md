# Events Reference

## Available Events

- TASK_CREATED - When a task is created
- TASK_COMPLETED - When a task is completed
- TASK_FAILED - When a task fails
- TASK_TIMEOUT - When a task times out
- BACKUP_CREATED - When a backup is created
- BACKUP_RESTORED - When a backup is restored
- LOG_WRITTEN - When a log is written
- SYSTEM_STARTED - When system starts
- SYSTEM_SHUTDOWN - When system shuts down
- ERROR_OCCURRED - When an error occurs

For detailed information, see documentation.
